# Code

This folder contains Stata scripts for data cleaning, variable construction, data merging, regression analysis, and figure generation.

## Scripts

- `01_clean_public_data.do`: cleans public oil price and demand data.
- `02_construct_variables.do`: constructs monthly time variables, log variables, and first differences.
- `03_merge_data.do`: merges datasets into a monthly analysis panel.
- `04_regression_analysis.do`: runs baseline log-level and first-difference regressions.
- `05_make_figures.do`: generates descriptive figures for the project.

## Software

The code is written in Stata.
