# Data Description

This project uses data from multiple sources.

## Public Data

- Oil prices are obtained from FRED.
- Monthly oil trade demand data are obtained from UN Comtrade / UNC database.

## Restricted Data

- Freight rates, fleet capacity, and fixtures data are obtained from Clarksons Research.
- These raw data are not publicly uploaded due to access restrictions.

## Variables

- Freight rate: tanker freight market condition.
- Demand: monthly oil trade / shipping demand.
- Fleet capacity: tanker supply, measured by fleet size or deadweight tonnage.
- Fixtures: signed spot contracts, used as a proxy for market activity and brokerage-related matching.
- Oil price: crude oil price used as a control variable.
